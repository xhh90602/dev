# 部署文档

## docker-compose文件

```yml
version: "3"
services:
  zhongwei-mobile-web-in-native:
    image: dev.hub.dztec.net/zhongwei/zhongwei-mobile-web-in-native
    restart: always
    container_name: zhongwei-mobile-web-in-native
    hostname: zhongwei-mobile-web-in-native
    environment:
     - TZ=Asia/Shanghai
    ports:
      - "8093:80"
    volumes:
      - ./moute-data/config.js:/www-data/static-mobile-mpa.config.js
      - ./moute-data/nginx.conf:/etc/nginx/nginx.conf
```
